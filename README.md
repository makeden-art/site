# Сайт coswickvrn

Современный веб-сайт для компании по производству паркета, созданный по образцу сайта Coswick.

## Структура проекта

```
├── index.html      # Главная страница
├── styles.css      # Стили сайта
├── script.js       # JavaScript функциональность
└── README.md       # Документация
```

## Особенности

- ✅ Современный адаптивный дизайн
- ✅ Плавная прокрутка и анимации
- ✅ Мобильное меню
- ✅ Секции: Каталог, Портфолио, Новости, О компании
- ✅ Интерактивные элементы
- ✅ Оптимизация производительности

## Запуск

Просто откройте `index.html` в браузере. Для локального сервера:

```bash
# Python 3
python -m http.server 8000

# Node.js (с http-server)
npx http-server
```

Затем откройте `http://localhost:8000` в браузере.

## Структура страницы

1. **Header** - Фиксированная навигация с выпадающими меню
2. **Hero** - Главный баннер с призывом к действию
3. **Collections** - Сетка коллекций паркета
4. **Mosaic** - Секция мозаичного паркета
5. **Portfolio** - Галерея реализованных проектов
6. **About** - Информация о компании
7. **Resources** - Полезные ресурсы для профессионалов
8. **News** - Новости, пресса, события
9. **Footer** - Контакты и дополнительная информация

## Кастомизация

### Цвета

Основные цвета определены в CSS переменных:

```css
--primary-color: #7A8B9A;
--primary-dark: #5A6B7A;
--dark-color: #1A1A1A;
```

### Шрифты

Используется шрифт Inter от Google Fonts. Можно изменить в `index.html`:

```html
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700&display=swap" rel="stylesheet">
```
